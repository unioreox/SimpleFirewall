# SimpleFirewall - Go实现的简易防火墙

## 目前实现的功能
- 基于iptables
- 目前功能仅限于用户验证和添加白名单
- 理论上可以提高节点安全性，抗封锁，降低主动探测风险
- 目前仅支持IPv4
